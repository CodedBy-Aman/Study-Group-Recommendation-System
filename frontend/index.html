Stable matching + skill complementarity

Gale-Shapley guarantees no two students would mutually prefer to swap groups. Our BidCalculator scores each pair by how well their skill vectors complement each other — weak areas matched against strengths.

// complementarity score
float computeBid(
  Student& a,
  Student& b
) {
  return dot(
    a.weaknesses,
    b.strengths
  );
}
